此项目是服务于10人规模的一家外贸公司。
ubuntu+mysql+python+django+ajax+JS+html+css
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
设想的功能包括：
    1，员工工作界面--录入信息，查询信息，发出任务，开始任务，任务提醒
    2，信息存储，挖掘--信息数据库存储，各式检索
    3，内部文档管理--撰写内部文档，添加评论
    4，与热门SNS和IM联通--短信微信提醒，公司微博发布
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
员工角色包括：
    1，销售--接受客户询价，对外报价，指派采购询价
    2，采购--接受销售的指派对工厂询价，监督工厂生产并按时交货
    3，单证
    4，财务
    5，总经理
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
